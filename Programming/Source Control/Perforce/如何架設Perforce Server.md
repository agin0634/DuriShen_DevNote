**Step.1 安裝 Helix Server(P4D)**
在 [Download Helix Core (P4D) | Perforce](https://www.perforce.com/downloads/helix-core-p4d) 選擇平台並下載

**Step.2 安裝設定**
開啟剛下載好的 helix-core-server-x64.exe。這邊照著預設即可，直接點擊 Next

設定 Port，基本上預設的 1666 即可
